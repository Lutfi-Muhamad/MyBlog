CSS (Cascading Style Sheets)

Selamat datang dalam petualangan dasar CSS, bahasa yang memberi warna dan gaya pada halaman web kita. Dalam perjalanan ini, kita akan memahami bagaimana CSS bekerja untuk mengubah tampilan situs web dari yang biasa menjadi luar biasa. 1.Pengantar CSS (Cascading Style Sheets)

Konsep Dasar
CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dan tata letak dokumen HTML. CSS memisahkan konten dari presentasi, memungkinkan pengembang web untuk mengontrol tampilan situs web secara efektif.

2. Fungsi dan Tujuan CSS

Pengaturan Tampilan
CSS memungkinkan pengembang web untuk mengatur tampilan elemen HTML seperti teks, gambar, dan layout halaman. Ini mencakup pengaturan warna, font, margin, padding, dan lainnya. Pemisahan Konten dan Presentasi
Dengan memisahkan konten (HTML) dari presentasi (CSS), pengembang dapat membuat perubahan pada tampilan situs web tanpa mempengaruhi struktur atau konten. Ini juga memudahkan pemeliharaan dan pembaruan situs web.

3.Komponen-Komponen Dasar CSS

Selektor
Selektor digunakan untuk memilih elemen HTML yang akan diberi gaya. Contoh selektor meliputi selektor tag, selektor kelas, dan selektor ID.
Properti dan Nilai
Properti CSS adalah aspek dari elemen yang ingin diatur (misalnya, warna, ukuran font), sedangkan nilai adalah pengaturan spesifik untuk properti tersebut. Contoh: color: blue; mengatur warna teks menjadi biru.
Deklarasi dan Aturan CSS
Deklarasi CSS adalah kombinasi dari properti dan nilai. Kumpulan deklarasi yang diterapkan pada selektor disebut aturan CSS. Contoh: p {
color: blue;
font-size: 16px;
}

4.Metode Penyisipan CSS

CSS Inline
CSS inline ditambahkan langsung ke elemen HTML menggunakan atribut style. Contoh:
<p style="color: blue;">Ini adalah teks biru.</p>

maka hasil dari code diatas akan merubah text menjadi warna biru seperti dibawah ini

Ini adalah teks biru.


CSS Inline
CSS internal ditambahkan di dalam tag 'style' di bagian 'head' dokumen HTML, seeprti berikut ini: 

                  <head>
                    <style>
                      p {
                        color: blue;
                      }
                    </style>
                  </head>
                  <p>Ini adalah teks biru.</p>
CSS Eksternal
CSS eksternal ditulis dalam berkas .css terpisah dan dihubungkan ke dokumen HTML menggunakan tag 'link', sebagai Contoh: 

                  <head>
                    <link rel="stylesheet" href="styles.css">
                  </head>

5.Penerapan CSS dalam Desain Web

Pengaturan Warna dan Latar Belakang
CSS memungkinkan pengaturan warna teks, warna latar belakang, gambar latar belakang, dan efek lainnya. Contoh:
body {
background-color: #f0f0f0;
}
h1 {
color: #333;
}

Tata Letak dan Grid
CSS digunakan untuk mengatur tata letak halaman menggunakan teknik seperti Flexbox dan Grid Layout. Contoh Flexbox: 

                  .container {
                    display: flex;
                  }
                  
                  .item {
                    flex: 1;
                  }

6.Tips dan Trik Dasar CSS

Penggunaan Kelas dan ID
Gunakan kelas (class) untuk mengatur gaya pada beberapa elemen dan ID (id) untuk elemen unik. Contoh:
<p class="highlight">Teks ini akan disorot.</p> <p id="unique">Teks ini unik.</p> 

              .highlight {
                background-color: yellow;
              }
              
              #unique {
                font-weight: bold;
              }

Pentingnya Konsistensi dan Komentar
Menjaga konsistensi dalam penamaan kelas dan penulisan kode CSS sangat penting untuk pemeliharaan yang mudah. Gunakan komentar untuk menjelaskan bagian kode yang kompleks. Contoh: 

                /* Gaya untuk tombol */
                .button {
                  background-color: #4CAF50;
                  color: white;
                  padding: 10px 20px;
                  border: none;
                }
