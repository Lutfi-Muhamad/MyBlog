Apa itu HTML?

HTML, atau HyperText Markup Language, adalah bahasa yang digunakan untuk membangun struktur dan konten halaman web. Dengan HTML, Anda bisa membuat website yang keren dan fungsional!

Gimana? udah paham kan? kalau belum, sini aku perjelas lagi, jadi HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web.

Jika diibaratkan sebuah tubuh, maka HTML merupakan kerangkanya. yaitu fondasi dasar dari sebuah tubuh Dalam membuat halaman web, HTML tidak sendirian. Ada bahasa lain lagi yang menjadi pelengkapnya, yakni CSS dan Javascript. mereka bertiga adalah gabungan untuk membuat sebuah website agar tampilannnya menjadi menarik
